# Security Policy Development Project

---

Bitra Immigration and Border Services Policy Development Project
This repository contains customized SANS security policy templates adapted for Bitra Immigration and Border Services, a fictional cybersecurity company.
The project demonstrates practical skills in security policy development, governance, documentation management.

--- 

These policies are tailored to the operational and regulatory requirements of an immigration and border service, focusing on the secure handling of:

- Passports
- Visa records
- Financial documentation

## 📌 Project Objectives

The main objectives of this project are:

- Adapt industry-standard SANS security policies to a specific organizational context
- Ensure proper management of sensitive immigration and border information
- Demonstrate structured security governance practices
- Provide clear policy templates that can be applied in a cybersecurity governance framework

---   

### Included Security Policies

The following policies are included in this repository:   
- Acceptable Use Policy – Defines how employees may use company IT resources responsibly
- Data Inventory Management Policy – Outlines inventorying, classification, and handling of sensitive data
- Database Credentials Standard – Establishes secure credential management for databases
- Internet Usage Standard – Provides guidance on safe and compliant internet access
- Network Device Management Policy – Covers configuration, monitoring, and maintenance of network devices
- Physical Security Management Policy – Defines access control, facility security, and protection of physical assets

### Compliance Alignment

This project aligns with industry best practices and regulatory frameworks, including:

- Activity	Compliance Framework	Purpose / Evidence
- Policy Development	SANS Security Policy Templates	Adapted and customized to the organization’s operational needs
- Data Protection Policies	ISO/IEC 27001: A.8 Asset Management, A.9 Access Control	Ensures proper handling and classification of sensitive immigration data
- Network & Device Policies	NIST 800-53 AC-2, AC-6, CM-2, SC-7	Establishes access controls and configuration management
- Physical & Access Policies	ISO A.11 Physical Security, NIST PE-2 / PE-3	Secures facilities and physical assets
- Documentation Version Control	NIST PL-2, ISO A.5 Information Security Policies
